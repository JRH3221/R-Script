Error Handling Documentation V1.0


Invalid Script Location

This error occurs when the compiler cannot open the script. This has many different causes but main being:

Invalid location, another program is using the script, the script requires higher privileges.


Must Use .rscript

The compiler reads the last 8 characters of the input location to test if they are ".rscript". If this is not the case then make sure that you have file extensions turned on, so you can change the file type to ".rscript".
